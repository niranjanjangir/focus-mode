Chrome extension to add a dark overlay above and below the cursor leaving a small ractangular space. I usually select a paragraph to read it comfortably due to other text and images on the webpage. This kind of allows me to focus on one paragraph at a time.
Early version as of now, will polish soon.

# How to use:
1. Clone the repository
2. Open `chrome://extensions`
3. Turn on the developer mode from the toggle
4. Click on "Load unpacked"
5. Select the folder containing the source code
6. Pin the extension and click on the icon to use it.
